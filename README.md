Chessboard using HTML & CSS
Project Overview
This project is a simple Chessboard layout built using HTML and CSS. It visually represents an 8×8 chessboard with alternating black and white squares and Unicode chess symbols to display pieces.
The purpose of this project is to demonstrate fundamental concepts of web layout, styling, and element positioning using CSS.
--- Features
  8×8 grid chessboard design using div elements, 
  Alternating black and white squares for a realistic look,
  Unicode chess symbols for each chess piece, 
  Simple, responsive design that fits neatly in the browser window, 
  Built purely with HTML and CSS — no JavaScript required ,
---Technologies Used
HTML5 — Structure of the page
CSS3 — Styling, colors, and layout
